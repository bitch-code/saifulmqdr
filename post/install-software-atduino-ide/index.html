Sun, Apr 4, 2021

Instal software Arduino IDE

Pada artikel kali ini kita akan melakukan penginstallan software arduino IDE (Integrated Development Environment). Sebelum mulai ke tutorual, kalian bisa siapin dulu bahan yg di perlukan:

  • Komputer/Leptop (Windows, Mac atau Linux).
  • Board arduino. (disini saya pakai arduino uno)
  • Kabel USB A-to-B.

Pertama pergi ke halaman Arduino download software dan dapatkan software Arduino versi terbaru yang sudah release dan stabil. pastiin versi yang didownload sesuai sama sistem operasi dikomputer kalian.

Download Arduino IDE

Auntuk proses installnyapun cukup mudah, tinggal kalian sesuaikan sama sistem operasi di komputer / leptop kamu. Berikut beberapa tip untuk membantu kamu.

Windows

Software arduino untuk versi windows punya 2 pilihan :

  1. Installer Kalian tinggal download lalu jalanin file exe nya untuk memulai instalasi kemudian tinggal ikutin aja langkah demi langkah sampe selesai.

  1. File Zip Apabila kamu memilih untuk mendownload versi zip file dari software Arduino. Kamu perlu mengekstrak file ke lokasi folder yang kamu pilih dan software Arduino langsung bisa digunain tanpa harus melakukan instalasi.

Jika kalian masih bingung, kalian bisa liat vidio ini:

Mac

Software Arduino untuk versi Mac hanya terdapat versi zip file. Setelah selesai mendownloadnya, kemudian double-click .zip file untuk mengekstraknya.

Selanjutnya kamu salin/copy Arduino application ke dalam folder Application untuk menyelesaikan proses instalasi.

Linux

Sebagai pengguna Linux enggak heran lagi kalau ada banyak versi Distro Linux, masing-masing dengan proses instalasi yang berbeda. Simak pada bagian Linux dari tutorial Instalasi Arduino disitus Arduino untuk beberapa link yang dapat membantu untuk bermacam-macam distribusi Linux.

Bagi pengguna Ubuntu sama Debian, instalasi dapat dilakukan dengan menjalankan command :

 
sudo apt-get update && sudo apt-get install arduino arduino-core
Dan untuk Distro Linux lainnya enggak jauh beda dari itu.

Upload Blink

Akhirnya sekarang waktunya membuka software Arduino. Kalian akan diperlihatkan dengan jendela interface terlihat seperti berikut:

Mari kita coba upload sebuah sketch Blink untuk memastikan bahwa setup Arduino benar-benar berfungsi. Pergi ke File menu pada Arduino, kemudian pergi ke Examples > 01.Basics > Blink untuk membukanya.

Sebelum kita dapat mengirimkan kode program kedalam Arduino, ada beberapa pengaturan yang perlu dilakukan.

  • Pilih tipe Board

Langkah ini diperlukan untuk memberitahu Arduino IDE, jenis board apa yang sedang kita gunakan. Pergi ke menu Tools. Kemudian sorot Board dan pastikan Arduino UNO telah dipilih.

  • Pilih Serial Port

Kemudian kita perlu memberi tahu Arduino IDE, serial port mana yang digunakan yang telah terhubung antara Arduino dengan komputer. Pergi ke Tools, kemudian sorot Serial Port dan pilih COM port yang terhubung dengan Arduino.

  • Upload!

Setelah selesai dengan pengaturan, Sekarang kamu siap untuk upload kode program! KLik ikon Upload (tanda panah kekanan) dan biarkan IDE melakukan compile dan upload kode kamu. Biasanya ini memerlukan waktu 10-20 detik untuk menyelesaikan proses tersebut. Ketika kode telah berhasil terisi kedalam Arduino, kamu akan melihat tampilan seperti ini:

Dan jika kamu melihat ke board Arduino, kamu akan melihat sebuah LED yang terhubung dengan pin 13 menyala selama satu detik, mati selama satu detik, menyala selama satu detik, dan terus seperti itu berulang-ulang. Jika kamu ingin mengatur waktu jeda ketika LED tersebut berkedip, coba untuk merubah nilai “1000” pada baris kode delay(1000);.

Ada sesuatu yang salah?

Uphs! Jika kamu tidak mendapatkan pesan “Done Uploading”, dan terdapat sebuah error, ada beberapa yang perlu kita periksa kembali.

Jika kamu mendapatkan keterangan error avrdude: stk500_getsync(): not in sync: resp=0x00 pada jendela console.

Kemungkinan pemilihan board dan serial port yang tidak cocok. Yakinkan tipe Arduino UNO telah dipilih (pada “Tools > Board” menu). Biasanya serial port merupakan penyebab yang sering terjadinya error, apakah serial port telah dipilih dengan benar (pada “Tools > Serial Port” menu)? Apakah Driver sudah berhasil terinstal? Untuk mengecek apakah benar serial port yang dipilih adalah yang terhubung dengan Arduino, bisa dilihat pada menu ketika boar sudah terhubung, kemudian lepaskan dan liaht serial port mana yang hilang. Apabila tidak ada port yang hilang, kamu perlu kembali untuk menginstal driver dengan benar.
